PaperCut MF

PaperCut is far and away the best print management product on the market. With deserved success, it is now used by more than 50,000 organizations across 100 countries. Monitor WA is the premiere distributor for PaperCut in Western Australia, so contact us today to find out what all the hype is about!

The PaperCut interface is simple and intuitive to use, which belies a powerful and extremely customizable print management engine that includes features such as:

  • Easy-to-use web interfaces for administration and user account management and tracking.
  • Print tracking, printing quotas and optional print charging.
  • Advanced reporting facilities.
  • Support for Windows, Mac, Linux and Novell network environments.
  • Embedded interfaces that run on many modern photocopiers and similar hardware.
  • VCC integration for print, copy and scan control supported by many modern photocopiers. VCC’s are particularly useful to provide a consistent interface for copier fleets that have a range of devices, or for specific devices that don’t support an embedded interface.
  • On-site account top-up via a Monitor WA Kiosk using cash or EFTPOS.
